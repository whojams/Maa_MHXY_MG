# 1.下载程序并设置

1.1下载对应系统（Linux、win、macos）版本的X86，arm程序，[下载地址](https://github.com/gitlihang/Maa_MHXY_MG/releases)<br>
1.2 软件主界面左下角设置→连接设置→触控模式，选择**MaaTouch**<br>

# 2.模拟器设置

1.1 最推荐MUMU模拟器（适配最好），兼容雷电模拟器，调整分辨率设置：**1280*720 240dpi**，其他16：9分辨率也能用。<br>
<details>
<summary>模拟器16：9分辨率</summary>
  1920x1080 (全高清，Full HD)<br>
  3840x2160 (超高清，4K)<br>
  2560x1440 (2K)<br>
  1280x720 (720P高清)<br>
</details>

1.2 显卡渲染模式选择**DirectX**模式，不要选择Vulkan模式.<br>

# 4.电脑设置
1.电脑缩放比例要求100%（影响文字识别），分辨率那个都可以，不影响使用。<br>
2.只有Windows电脑需安装[vc_redist](https://aka.ms/vs/17/release/vc_redist.x64.exe)点击前往下载地址<br>
3.需安装[.NET 10](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0)点击前往官方下载地址
